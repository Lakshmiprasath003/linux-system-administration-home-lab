# Linux System Administration Home Lab

## Overview

This project demonstrates a Linux System Administration Home Lab built using WSL (Windows Subsystem for Linux) and Ubuntu. The objective was to gain practical experience in Linux administration tasks commonly performed by system administrators and IT support engineers.

## Technologies Used

* WSL2
* Ubuntu Linux
* OpenSSH Server
* Samba
* Linux Command Line

## Tasks Performed

* Installed Ubuntu using WSL on Windows.
* Updated and upgraded Linux packages.
* Practiced Linux file and directory management.
* Created and managed Linux users and groups.
* Configured file ownership and permissions using `chmod` and `chown`.
* Enabled and tested SSH for secure remote administration.
* Configured shared folder access between Windows and Ubuntu.
* Installed and configured a basic Samba file server.
